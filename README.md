# Dou.ua Jobs&Events
This Android app lists jobs and events from rss feed of Dou.ua, which is a popular ukrainian internet portal related to software development and information technology.

### Language:
Kotlin

### Architecture: 
MVP

### Libraries:
* Retrofit2
* RxJava
* Room
* Jsoup
* Picasso
